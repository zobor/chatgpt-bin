# chatgpt-bin

基于 chatgpt 官方Open API的命令行，解决官网IP限制、连接不稳定等体验问题。

当然也有在线免费的版本，支持Web和H5，地址：[https://www.bproxy.cn/chat](https://www.bproxy.cn/chat)

## 如何使用
```bash
npm i -g chatgpt-bin

> chatgpt
> 我: $config.proxy=xxx
> 我: $config.key=yyy
> 我: 输入你的问题
```
> 安装完之后，启动 chatgpt。需要先配置代理和你的ApiKey。配置完成之后就可以提问了。
